# 📧 AI Gmail Reply Assistant  

An **AI-powered email agent** that helps you generate **polite and professional replies** to incoming emails.  
Built with **Python, Streamlit, and Together AI**, this app allows you to paste an email and instantly get a well-written reply.  

🔗 Live Demo (optional if you deploy): *coming soon*  

---

## ✨ Features  
- 📝 Paste any incoming email and get an instant AI-generated reply.  
- 🤖 Powered by Together AI’s large language models (Mixtral).  
- 🎨 Clean and responsive interface built with Streamlit.  
- 🔐 Secure API key handling via `.env` file.  
- ⚡ Lightweight and easy to run locally.  

---

## 🛠️ Tech Stack  
- **Frontend/UI:** Streamlit  
- **Backend Logic:** Python  
- **AI Model:** Together AI (Mixtral-8x7B-Instruct)  
- **API Handling:** `requests`  
- **Environment Management:** `python-dotenv`
